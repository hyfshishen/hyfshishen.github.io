Project on Sci. & Tech. resource pool based on Knowledge Graph.

Apr 27, 2019
Project interface with demo data, screenshot by Yafan Huang

Background

In Hubei, Hunan and Jiangxi province of China, there are massive of science and technology resource data. Take Hubei for example, 129 universities in Hubei creates plenty of papers, patents and etc. every year. On the one hand, these data belong to different technology departments and cause information barriers for users. On the other hand, these data management systems are less intelligent, which indicates they show poor performance in processing large amounts of data and reasoning.

Purpose

Aiming to build a information management system that includes all the science and technology resource (such as papers, patents, etc.) of Hubei, Hunan and Jiangxi province of China. This project requires: (1) a visualization interface via Knowledge Graph, (2) some intelligent data analysis functions.

Method

(1) Project framework via maven + sprintMVC.
(2) Create and apply Knowlege graph engine via Java.
(3) Design and create web-interface via Bootstrap-based HTML and CSS.
(4) Visualize knowledge graph via vis.js-based JavaScript.
(5) Data analysis algorithms written via Java, visualized via ECharts and Chart.js-based JavaScript .
(6) Project deployed on a Linux server.
